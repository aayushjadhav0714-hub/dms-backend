Disaster Rescue Management System 🛡️
A Full-Stack Web Application designed to enhance disaster preparedness and response by connecting citizens with administrative authorities in real-time.

 Live Demo
Check out the live project here: https://dsm-project.netlify.app/

 Project Overview
The Disaster Rescue Management System is a platform built to tackle natural calamities like floods, earthquakes, and heavy rainfall. It provides a dual-interface system (User and Admin) to streamline communication and resource allocation during emergencies.

 Key Features
 User Panel
Real-time Weather Dashboard: Integrated with OpenWeather API to display live temperature and humidity for early warning.

Disaster Reporting: Users can report incidents by providing the disaster type and specific location.

Safety Guides: Comprehensive 'Do's and Don'ts' for various natural disasters.

Emergency Contacts: Quick access to essential helpline numbers (Police, Fire, Ambulance).

 Admin Panel
Centralized Dashboard: A dedicated interface for authorities to monitor all incoming disaster reports.

Data Management: Efficiently manage and track rescue requests stored in the database.

Actionable Insights: Helps in quick decision-making and resource deployment based on real-time data.

 Tech Stack
Frontend: HTML5, CSS3, JavaScript (Responsive UI)

Backend: Node.js, Express.js

Database: MySQL

API: OpenWeatherMap API

Deployment: Netlify (Frontend)

 Installation & Setup

Bash
